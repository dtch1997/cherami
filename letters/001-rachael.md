# 001 — to Rachael (resuming French)

**Date:** 2026-06-29

## Draft (what I wanted to say)

Bonjour Rachael! [I have resumed my French lessons.] [I am using Claude to do
it.] [It is going well.]

## Correction

> Bonjour Rachael !
> J'ai repris mes cours de français. J'utilise Claude pour ça. Ça se passe bien.

### Notes

- **Bonjour Rachael !** — In French, the marks `! ? : ;` take a space *before*
  them. So `Rachael !`, not `Rachael!`.
- **J'ai repris mes cours de français.** — "I have resumed…" is the *passé
  composé*: present of **avoir** + a past participle. `reprendre` (to resume) is
  irregular, like `prendre` → `pris`, so its participle is **repris**.
  `mes cours` = "my lessons" — `mes` because the noun is plural.
- **J'utilise Claude pour ça.** — `utiliser` = to use (regular). `pour ça` =
  "for that"; `pour` + a noun or an infinitive expresses purpose.
- **Ça se passe bien.** — Set phrase for "it's going well." Literally "it
  happens itself well," from `se passer` (to happen / go).
